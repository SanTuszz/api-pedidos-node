# API de Pedidos

API REST desenvolvida em Node.js utilizando Express.

## Tecnologias
- Node.js
- Express

## Funcionalidades
- Criar pedido
- Listar pedidos
- Buscar pedido por ID
- Atualizar pedido
- Remover pedido

## Rotas

POST /orders  
Cria um novo pedido.

GET /orders  
Lista todos os pedidos.

GET /orders/:id  
Busca pedido por ID.

PUT /orders/:id  
Atualiza pedido.

DELETE /orders/:id  
Remove pedido.

## Executar projeto

npm install  
node index.js

Servidor roda na porta 3000.
