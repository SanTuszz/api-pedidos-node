# API de Pedidos

API REST desenvolvida em Node.js com Express para gerenciamento de pedidos.

## Rotas

POST /order
Cria um pedido.

GET /order/:id
Busca um pedido.

GET /order/list
Lista todos os pedidos.

PUT /order/:id
Atualiza um pedido.

DELETE /order/:id
Remove um pedido.

## Rodar o projeto

npm install
node index.js
